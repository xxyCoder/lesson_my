# 数组
- 创建数组的时候可以给构造函数一个参数
    - 参数是一个数值表示数组长度
    - 其他值则是内容
- from
    - 可以将任何类似数组的结构转换为数组
    - 第一个参数是类数组
    - 第二个参数是映射关系
    - 第三个参数是替换this，对于第二个参数写成箭头函数没影响
- to
    - 可将一组参数转换为数组实例
- length不是只读属性，可以通过修改其值，实现对数组增删
